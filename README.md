# Office Diorama 3JS

A 3D office diorama built with Three.js, showcasing a detailed office scene with baked textures and interactive orbit controls.

## Features

- **3D Office Model**: Loads a GLTF model of an office environment.
- **Baked Textures**: Uses pre-computed textures for optimized performance.
- **Orbit Controls**: Allows users to rotate, zoom, and view the scene from different angles.
- **Responsive Design**: Adapts to different screen sizes and devices.

## Technologies Used

- [Three.js](https://threejs.org/) - 3D graphics library
- HTML5 Canvas
- CSS3
- GLTF Loader for 3D models

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/OfficeDiorama3JS.git
   ```
2. Open `index.html` in your web browser.

No additional setup required; all dependencies are loaded from CDNs.
